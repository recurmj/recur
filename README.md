# Recur Canon Papers

This repository hosts the canonical Recur papers:

- **Recur Whitepaper — “The Flow Layer for Digital Value”**
- **Recur Technical Paper — “Defining the Permissioned-Pull Standard for Digital Value”**

It is an archival repository for the documents themselves, not the reference
implementation (which lives in a separate code repo).
